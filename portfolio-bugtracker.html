Project Motive

The Bug Tracking Application was developed to provide a customizable platform for managing bugs and issues, inspired by JIRA but with unique features. It aims to improve team collaboration by allowing users to report bugs, managers to oversee tasks, developers to resolve issues, and administrators to manage the system. The addition of a file upload system enhances the application by enabling users to attach relevant documents or images to issues.

Project Scope

The Bug Tracking Application, built with the MERN stack, supports four key roles: users, managers, developers, and administrators. It allows users to report bugs, managers to manage tickets and projects, developers to create and resolve issues, and administrators to oversee the system. The application also includes a file upload feature to attach documents or images to issues, enhancing bug tracking and project management.
