Project information

  • Project Name:

    Stadium Seat Booking System

  • Category: Web design
  • Client: Audience / Visiter / Manager
  • Project date: March, 2022
  • Project URL: Repo Link
  • Django, HTML, CSS, JQuery, MYSQL

Project Motive

The purpose of the project is to create a system which makes it easier for the ticket purchaser, select a seat that they want. This modern system will make it easier to organize the stadium.

Project Scope

The aim of this project is to create a web application for managing ticket reservations and make it easier for the user to book seat of his/her choice.

Profile

  • Administrator

    Organize and manage the seat allotment made by users, keep a check on the vacant seat and cancelation seat and list of booked seat and keep a track on guard department and other departments, finances , inventory etc.

  • User Profile

    User can check their booked seat and they are able to cancel their seat.
